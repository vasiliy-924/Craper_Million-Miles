Тестовое задание

Цель: проверить навыки кандидата в веб-скраппинге и веб-разработке на реальной задаче.

Задачи:
1. Парсинг данных с сайта CarSensor (https://carsensor.net/) воркером:
• собрать информацию: марка, модель, год, пробег, цена, фото и т.д.
Важное уточнение: данные на сайте представлены в основном на японском языке. Необходимо создать словарь для перевода и нормализации данных.
• настроить скраппинг 1 раз в час.
• сохранение в базу данных.
2. Backend:
• Простой backend с двумя эндпоинтами - отдача списка автомобилей с фильтрами, сортировкой и пагинацией, а также JWT авторизация.
3. Веб-приложение для просмотра содержимого БД:
• развернуть простое веб-приложение (NextJS/React) с просмотром ассортимента автомобилей, добавленных в БД.
• настроить вход по логину+паролю через JWT (log:pass - admin:admin123).
• необходимо отображать список автомобилей + сделать страницу для каждого автомобиля с детальными характеристиками.

Требования:
• адаптивеная верстка под ПК и мобильные устройства
• быстрый отклик сайта

Формат сдачи:
• ссылка на исходный код (GitHub/GitLab/другая платформа);
• ссылка на развернутое веб-приложение.

Дедлайн: 2 дня с момента получения задания.
